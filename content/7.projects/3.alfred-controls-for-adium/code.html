A set of AppleScripts so you can set your chat status from Alfred App. Remember, you will need Alfred and the Powerpack to use this.

Adium Controls for Alfred makes it super easy to set an away message and start your screen saver. Personally I have my computer lock when the screen saver goes up so it quickly secures my computer as I walk away. It's perfect for letting other people know if you'll be back in a minute (brb) or back in an hour (lunch).

Source, downloads and full readme is up on GitHub.
